MVP Workflow

AI Task Orchestrator

Task listesi gir, görevleri parçala, prompt üret, prompt geçmişini ve approval akışını tek panelden izle.

Provider bilinmiyor

Provider durumu henüz yüklenmedi.

1. Fikir Girişi

Fikrinizi girin, sistem otomatik olarak projelendirsin, görevlere bölsün ve promptları üretsin.

2. Görevler

Görev seçildiğinde prompt üretimi ve approval simülasyonu bu karttan tetiklenir.

Henüz görev üretilmedi.

3. Seçili Görev

Seçilen görevin özeti.

Henüz görev seçilmedi.

4. Son Prompt

Prompt Generator çıktısı.

Henüz prompt üretilmedi.

5. Prompt Geçmişi

Üretilen prompt çalıştırma kayıtları.

Henüz prompt geçmişi yok.

6. Approval

Temel approval akışı için bekleyen işlemler.

Henüz onay bekleyen işlem yok.

7. Loglar

Süreç akışı ve önemli backend olayları.

Henüz log yok.

Prompt Kuyruğu

Görevlerden üretilen promptları inceleyin, onaylayın veya reddedin. Onaylıları kuyruğa alıp sırayla yürütün.

Henüz prompt üretilmedi. Yukardan fikrinizi girip 'Fikirden Prompt Üret' butonuna tıklayın.
